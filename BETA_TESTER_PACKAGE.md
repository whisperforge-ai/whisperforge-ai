# WhisperForge AI - Beta Tester Package

## Welcome

Thank you for helping test WhisperForge AI.

WhisperForge is an AI creative workspace designed to help audio creators build characters, write scripts, and develop ongoing stories.

---

# What We Want Feedback On

## 1. Character Studio

Try creating a character.

Tell us:

- Was it easy?
- Did the character feel unique?
- What information was missing?

---

## 2. Script Studio

Try creating a story idea.

Tell us:

- Would this save you time?
- Did the output feel usable?
- What would improve it?

---

## 3. Series Studio

Try planning multiple episodes.

Tell us:

- Would this help you stay consistent?
- What features would you add?

# Feedback Questions
