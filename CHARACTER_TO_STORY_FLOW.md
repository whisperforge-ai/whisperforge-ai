# WhisperForge AI - Character To Story Flow

## Purpose

Create a natural path from a finished character into a story.

---

# Starting Point

Creator finishes a character.

Example:

Viktor Vale

---

# Next Prompt

WhisperForge asks:

"Your character is ready. What would you like to create next?"

Options:

🎬 Create a Scene

Create a single moment featuring this character.

---

📖 Start a Story

Build an episode, series, or larger narrative.

---

🌍 Expand The World

Create locations, history, and important details.

---

# Scene Creation

Uses character information:

- Personality
- Voice
- Background
- Motivations

Creator adds:

- Situation
- Location
- Mood
- Conflict

---

# Story Creation

Uses:

- Existing characters
- Character relationships
- World information

Creates:

- Series idea
- Episode ideas
- Story direction

---

# Goal

The creator should move naturally:

Character → Scene → Story → Series

Without rebuilding information.

---

# Experience Principle

Every creation should unlock the next creative step.
