# WhisperForge AI - Creator Memory System

## Purpose

Help creators maintain consistency across characters, stories, and worlds.

WhisperForge should remember creative decisions so creators don't have to search through old notes.

---

# Character Memory

WhisperForge remembers:

## Identity

- Name
- Age
- Character type
- Appearance
- Occupation

## Personality

- Strengths
- Weaknesses
- Fears
- Motivations
- Values

## Voice

- Speaking style
- Tone
- Common phrases
- Emotional patterns

## Relationships

- Allies
- Enemies
- Important connections
- Relationship changes

## Character Growth

Tracks:

- Major decisions
- Lessons learned
- Emotional changes
- Important events

---

# Story Memory

WhisperForge remembers:

- Episode summaries
- Major events
- Unresolved conflicts
- Important discoveries
- Future story threads

---

# World Memory

WhisperForge remembers:

- Locations
- Rules
- History
- Important objects
- Groups or factions

---

# Creator Notes

Creators can add:

- Private ideas
- Future plans
- Things to avoid
- Inspiration notes

---

# Memory Goal

A creator should be able to return months later and continue creating without rebuilding everything from scratch.

---

# Guiding Principle

WhisperForge should remember the story so the creator can focus on telling it.
