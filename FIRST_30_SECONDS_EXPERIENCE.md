# WhisperForge AI - First 30 Seconds Experience

## Goal

A new user should understand WhisperForge within seconds and know their first action.

---

# Current Experience

User opens app.

Sees:

- Character list
- Add character button
- Sample characters

---

# Current Problem

The user does not immediately understand:

- What WhisperForge does
- How characters connect to stories
- Where to begin

---

# Desired Opening

## Welcome Message

"Welcome to WhisperForge.

Turn your ideas into characters, stories, and worlds.

Not sure where to start? We'll help you build it."

---

# Primary Actions

## Create

Start a new creation.

Options:

🎭 Character

Create someone new.

✍️ Story

Build a scene or episode.

💡 Idea

Start with a simple thought.

---

# Secondary Actions

Explore:

- Inspiration Library
- Existing creations
- Templates

---

# Success Moment

Within the first session, the user should:

1. Create something
2. Understand what they created
3. Know what to do next

---

# Experience Principle

Never leave a new creator wondering:

"What now?"
