# WhisperForge AI - First Time User Flow

## Goal

A new user should understand WhisperForge within 2 minutes.

---

# Step 1: Welcome

User sees:

"Welcome to WhisperForge AI"

Message:

Create characters, write immersive scripts, and build ongoing audio stories from one creative workspace.

---

# Step 2: Choose Starting Point

Options:

🎭 Create a Character

Start by building a unique personality, voice, and backstory.

---

✍️ Create a Script

Turn an idea into a scene or episode.

---

📚 Build a Series

Develop an ongoing story universe.

---

# Step 3: Character Creation

User creates:

- Name
- Character type
- Personality
- Voice style
- Background
- Story possibilities

---

# Step 4: First Creation

User generates:

- Episode idea
- Scene setup
- Character dialogue concept
- Performance notes

---

# Step 5: Save & Continue

User can:

- Save project
- Create another episode
- Expand the story

---

## Success Metric

The user should finish their first creative project during their first session.
