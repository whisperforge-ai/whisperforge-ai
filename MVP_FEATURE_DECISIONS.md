# WhisperForge AI - MVP Feature Decisions

## Core Features (Build First)

These are required for the first usable version.

## Character Studio

Purpose:
Help creators build consistent characters.

Includes:

- Character name
- Type
- Personality
- Voice style
- Voice traits
- Speaking style
- Background
- Creator notes

---

## Script Studio

Purpose:
Help creators turn characters into stories.

Includes:

- Character selection
- Scenario
- Mood
- Length
- Episode concept
- Scene setup
- Performance notes

---

## Story Series Planning

Purpose:
Help creators continue building worlds.

Includes:

- Series title
- Episode ideas
- Character connections
- Story progression

---

## Library

Purpose:
Keep creations organized.

Includes:

- Saved characters
- Saved scripts
- Saved series

---

# Features To Delay

## Marketplace

---

## Analytics

Reason:
Useful later after users exist.

---

## Collaboration Tools

Reason:

---

## Mobile App

Reason:
Validate the product before investing in development.

---

# MVP Goal

A creator should be able to:

Create a character → Generate a story concept → Save it → Continue building.
