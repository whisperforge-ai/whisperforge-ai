# WhisperForge AI - Screen Map Version 1

## Purpose

Define the structure of the first usable version of WhisperForge.

The goal is to create a clear journey:

Idea → Character → Story → Script → Saved Creation

---

# Screen 1: Home / Forge Entrance

## Purpose

Welcome creators and explain what WhisperForge does.

## Shows:

Welcome message:

"Welcome to WhisperForge.

Turn your ideas into characters, stories, and worlds."

## Main Actions:

🎭 Create Character

✍️ Create Story

💡 Get Inspiration

## Secondary Sections:

- Continue Creating
- My Characters
- My Stories
- Inspiration Library

---

# Screen 2: My Characters

## Purpose

Display the creator's original characters.

## Shows:

Character cards:

- Image
- Name
- Character Type
- Short Description
- Last Updated

## Actions:

- Open Character
- Edit Character
- Create Story With Character

---

# Screen 3: Character Creation

## Purpose

Guide users through creating a character.

## Sections:

## The Spark

"What idea do you have?"

## Identity

- Name
- Character Type
- Age
- Occupation

## Personality

- Traits
- Strengths
- Weaknesses
- Motivations

## Voice

- Voice Style
- Voice Traits
- Speaking Style

## Story Foundation

- Background
- Story Possibilities
- Creator Notes

---

# Screen 4: Character Hub

## Purpose

Turn a saved character into a creative starting point.

## Shows:

- Character summary
- Personality
- Voice
- Background

## Actions:

🔥 Forge a Scene

📖 Start a Story

🌍 Expand World

📚 View Scripts

---

# Screen 5: Script Forge

## Purpose

Create scenes and episodes.

## Fields:

- Script Title
- Characters Involved
- Location
- Mood
- Scenario
- Conflict
- Dialogue Style
- Creator Notes

---

# Screen 6: Story Library

## Purpose

Organize finished and unfinished creations.

## Shows:

- Stories
- Scripts
- Drafts
- Completed Projects

---

# Screen 7: Inspiration Library

## Purpose

Help creators overcome the blank page.

## Contains:

- Sample Characters
- Templates
- Story Prompts

## Action:

"Create Your Own Version"

---

# Design Principle

Every screen should answer:

"What can I create next?"
