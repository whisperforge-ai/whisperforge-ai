# WhisperForge AI - Script Forge Design

## Purpose

Help creators transform characters into scenes, episodes, and stories.

---

# Character Forge Answers

"Who is this?"

Contains:

- Identity
- Personality
- Voice
- Background
- Creator notes

---

# Script Forge Answers

"What happens?"

Contains:

- Script title
- Characters involved
- Location
- Mood
- Scenario
- Conflict
- Dialogue direction
- Creator notes

---

# Connection

Scripts should pull information from saved characters.

The creator should not have to rebuild character information every time.

---

# Goal

A creator should be able to select a character and quickly create a scene that feels consistent with that character.
