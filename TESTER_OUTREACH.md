# WhisperForge AI - Tester Outreach

## Goal

Find the first creators willing to give feedback on the WhisperForge AI concept.

---

## Message

Hi!

I'm building WhisperForge AI, a creative tool designed for audio roleplay creators.

It helps creators:

- Build detailed characters
- Create consistent voices and personalities
- Generate script ideas
- Plan ongoing story series

I'm looking for a few creators willing to look at the early prototype and give honest feedback.

I'm not selling anything right now — I’m trying to make sure I build something creators would actually use.

Would you be interested in taking a look?

---

## Questions For Testers

1. What type of audio content do you create?

2. What takes the most time when creating content?

3. Which feature would help you the most?

4. Would you use a tool like this weekly?

5. What would make this worth paying for?
